# Atlaix Bundle Detector

A powerful tool for detecting token bundles and analyzing wallet behavior.

## Features
- Recursive wallet tracing
- Multi-chain support (Solana, Ethereum, BSC, etc.)
- Risk scoring and forensic analysis
- Real-time volume and bundle detection

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
